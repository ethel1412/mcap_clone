Use this page to assign real-world dates to the Master Roadmap for an upcoming admission cycle. Please ensure the following:

  • Overlapping Dates: Schedule steps can overlap with each other, giving you flexibility to run multiple tasks simultaneously.
  • The Window Partition: Steps cannot overlap with the Student Application Window itself.
  • Pre-Admission Limits: Institute setup and controller approval steps must be completed before the application window opens.
  • Counselling Limits: Seat allotment and payment rounds cannot begin until after the application window closes.
